# C-mo-funciona-el-Frontend-de-OnchainKMS
Cómo funciona el Frontend de OnchainKMS
Un recorrido técnico por la arquitectura, flujos internos y componentes clave de la aplicación.

Tu frontend de OnchainKMS es una aplicación moderna desarrollada con React, optimizada para funcionar tanto como web completa como en el modo Farcaster Miniapp. Combina UI dinámica, conexión con APIs, integración con Strava y acciones blockchain (mint), todo dentro de una estructura modular y clara.
